Turbine Help

USAGE

     turbine [<opts>*] <program> [<args>*]

DESCRIPTION

     Run a Turbine program.

     By default <program> is assumed to be a Tcl (*.tic) script,
     normally generated by STC, but if -x is
     specified, an executable can be provided instead.

     Swift/T Web Site: http://swift-lang.org/Swift-T
