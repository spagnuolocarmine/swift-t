TURBINE RUNTIME ENVIRONMENT

   ADLB_SERVERS
                         number of ADLB servers

   TURBINE_LOG
                         If 1, enable log

   TURBINE_DEBUG
                         If 1, enable Turbine debugging
                         (if enabled at configure time)

   ADLB_DEBUG
                         If 1, enable ADLB debugging
                         (if enabled at configure time)

   TURBINE_LAUNCH_OPTIONS
                         provide additional arguments to mpiexec

   See the web site for more environment variables
